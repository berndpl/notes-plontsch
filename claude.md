# Blog with Notes

+ Using Jekyll 
+ Hosted on Github
+ For colors I am using the catpuccin color theme

### Posts
- I am collecting notes saved as indiviual posts in the /_posts/ folder
- Never edit or change anything in this posts folder

### Generated Pages

Everytime I add a new posts the pages for questions and insights need to be re-generated.

- The page description containes anote that the respective pages was generated. when and which model was used to    
  genrate it.
- Each page shows extracted content as cards 
- each card can be based on one or more posts
- link back to these posts
- Before generating, read all the posts but don't change anything

Card content
A simple card structure
- A title
- Click anywhere in the card linking back to the source post
